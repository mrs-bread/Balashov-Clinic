# Balashov-Clinic 🩺
**Студент:** Балашов Д. В.  
**Группа:** МО-32/1  
**Тема:** Разработка веб-системы для автоматизации управления приймами и ведения медицинских карт медицинского центра
---
### О проекте

Веб-система для автоматизации управления приёмами и ведения электронных медицинских карт в амбулаторной медицинской организации (поликлиника / медицинский центр).

**Цель проекта** - разработка удобного, безопасного и соответствующего законодательству Российской Федерации (Федеральный закон № 152-ФЗ) решения, направленного на:

- сокращение рутинных операций регистратуры  
- минимизацию ошибок при планировании приёмов  
- обеспечение оперативного доступа врачей к истории пациента  
- предоставление пациентам удобного личного кабинета
---
### Основная функциональность

- 🔐 **Регистрация и аутентификация** пользователей с разграничением прав по ролям  
  (администратор, регистратор, врач, пациент)
- 👨‍⚕️ **Управление справочниками** врачей, специализаций, кабинетов
- 📅 **Формирование расписаний** и автоматическая генерация временных слотов с проверкой конфликтов
- 🗓️ **Запись на приём**  
  - через регистратуру  
  - через личный кабинет пациента
- 🔄 **Управление статусами записей**  
  (запланировано → подтверждено → отменено → завершено)
- 📋 **Электронная медицинская карта**  
  - фиксация визитов  
  - версионирование записей  
  - история посещений
- 👤 **Личный кабинет пациента**  
  - просмотр предстоящих приёмов  
  - история визитов
- 📊 **Аналитика и отчётность** для администратора  
  (загруженность врачей, количество визитов, динамика обращений)
