# Balashov-Clinic

Веб-система автоматизации управления приёмами и ведения медицинских карт медицинского центра.

## Краткое описание проекта

Учебный проект, разрабатываемый в рамках дисциплины «Программная инженерия» (группа МО-32/1, Кубанский государственный университет).  
Цель — создание веб-приложения для автоматизации ключевых процессов амбулаторной медицинской организации с соблюдением требований к защите персональных данных.

## Основная функциональность

- Регистрация и аутентификация пользователей с разграничением прав по ролям (администратор, регистратор, врач, пациент)
- Управление справочниками врачей, специализаций и кабинетов
- Формирование расписаний, автоматическая генерация временных слотов, проверка конфликтов
- Запись на приём (регистратурой и пациентом через личный кабинет)
- Управление статусами записей (запланировано, подтверждено, отменено, завершено)
- Ведение электронной медицинской карты с фиксацией визитов и версионированием записей
- Личный кабинет пациента (просмотр предстоящих приёмов и истории визитов)
- Базовая аналитика и отчётность для администратора (загруженность врачей, количество визитов)

## Технологический стек

**Backend**  
- PHP 8.1+  
- Laravel (фреймворк)  
- MariaDB 10.11 (СУБД)  

**Frontend**  
- Vue.js 3  
- Bootstrap 5  
- Chart.js (визуализация отчётов)  

**Инструменты**  
- MySQL Workbench — моделирование базы данных  
- Git + GitHub — контроль версий
